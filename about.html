Опис предметної області

Платформа книжкового онлайн магазину

BookStore Pro - це платформа цифрового книжкового магазину, розроблена для революційних змін у способі, яким люди відкривають, купують та взаємодіють з книгами. Платформа служить мостом між традиційною книжковою торгівлею та сучасною цифровою комерцією, пропонуючи користувачам інтуїтивний та захоплюючий досвід для всіх їхніх потреб у читанні.

Основна функціональність

Платформа охоплює кілька ключових функціональних областей:

  • Каталог книг: Зручний список книг з детальними даними, включаючи назви, авторів, жанри, ціни та рейтинги.
  • Облікові записи: Реєстрація, вхід, редагування профілю та зміна паролю.
  • Пошук та фільтрація: Пошук за назвою/автором, фільтри за жанром, рейтингом, ціною та сортування.
  • Кошик покупок: Додавання товарів, перегляд кошика та імітація оформлення замовлення.
  • Відгуки та рейтинги: Перегляд рейтингів книг та можливість залишити власний відгук (для авторизованих користувачів).
  • Список обраного: Можливість зберігати улюблені книги у власному профілі для швидкого доступу.

Опис бізнес-логіки

Архітектура та збереження даних

BookStore Pro працює як клієнтський веб-застосунок, де вся бізнес-логіка виконується у браузері користувача. Замість серверної бази даних використовується LocalStorage браузера та статичні дані.

1. Реєстрація та авторизація

  1. Користувач вводить дані (email, пароль, ім'я) у форму реєстрації.
  2. Скрипт перевіряє валідність даних та чи не існує вже такого користувача в LocalStorage.
  3. Створюється об'єкт користувача, який зберігається у LocalStorage під ключем 'users'.
  4. При вході дані звіряються зі збереженими. Успішний вхід створює сесію ('currentUser').

2. Каталог та пошук

  1. Список книг завантажується зі статичного масиву даних (JavaScript об'єктів).
  2. Пошук та фільтрація відбуваються миттєво шляхом перебору масиву книг у браузері.
  3. Фільтри (ціна, жанр, рейтинг) застосовуються комбіновано, динамічно оновлюючи DOM-дерево сторінки.

3. Робота з кошиком

  1. Користувач додає книгу до кошика - ID книги та її кількість записуються в масив 'cart' у LocalStorage для поточного користувача.
  2. При завантаженні сторінки скрипт зчитує цей масив та відображає товари.
  3. Видалення товару оновлює масив у сховищі та миттєво перераховує загальну суму.
  4. "Оформлення замовлення" імітує процес покупки, очищуючи кошик користувача після підтвердження.

4. Відгуки та рейтинги

  1. Авторизований користувач може залишити відгук.
  2. Відгук зберігається в LocalStorage, прив'язуючись до ID книги.
  3. Рейтинг книги перераховується як середнє арифметичне всіх збережених оцінок.
  4. Дані оновлюються динамічно без перезавантаження сторінки.
