1. Селектори тегів

Статус: Виконано

У проєкті активно використовуються селектори тегів для стилізації базових HTML елементів.

HTML-код з використанням тегів:

<header>
    <nav>
        <div class="logo">
            <h1>BookStore Pro</h1>
        </div>
        <ul class="nav-links">
            <li><a href="#home">Головна</a></li>
            <li><a href="#books">Книги</a></li>
        </ul>
    </nav>
</header>

<main>

        <h2>Welcome!</h2>

</main>

4. Інші селектори

Статус: Виконано

Використовуються сусідні, дочірні селектори, селектор атрибута та універсальний селектор.

Універсальний селектор:

/* Універсальний селектор */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Дочірні селектори:

/* Дочірні селектори */
.nav-links a {
    text-decoration: none;
    color: #000000;
    font-weight: normal;
    font-size: 18px;
    transition: color 0.3s ease;
}

.footer-section ul li a {
    opacity: 60%;
    color: white;
    text-decoration: none;
    font-size: 14px;
    transition: color 0.3s ease;
}

Сусідні селектори:

/* Сусідні селектори */
.hero-text h2 + h3 {
    font-size: 20px;

    font-weight: normal;
}

.newsletter h2 + h3 {

    margin-bottom: 30px;
    font-weight: bold;
}

Селектори атрибутів:

6. Стильове оформлення резюме

Статус: Виконано

Сторінка about.html містить детальний опис проекту з професійним стильовим оформленням.

Основні характеристики резюме:

  • Структурований контент: Розділи з описом предметної області, бізнес-логіки, функціональних та нефункціональних вимог
  • Професійне оформлення: Використання існуючих стилів проекту для консистентності
  • Адаптивний дизайн: Підтримка різних розмірів екранів
  • Чітка типографіка: Легко читабельний текст з правильним ієрархічним структуруванням
  • Навігація: Зручні посилання для повернення на головну сторінку

HTML-код резюме:

<div class="about-content">
    <a href="index.html" class="back-link">← Повернутися на головну</a>
    
    <div class="section">
        <h2>Опис предметної області</h2>
        <h3>Платформа книжкового онлайн магазину</h3>
        <p>BookStore Pro - це платформа цифрового книжкового магазину...</p>
        
        <h3>Основна функціональність</h3>
        <ul>
            <li><strong>Каталог книг:</strong> Зручний у використанні список книг...</li>
            <li><strong>Система облікових записів:</strong> Реєстрація, автентифікація...</li>
        </ul>
    </div>
</div>
