# BookStore Pro

BookStore Pro - це сучасний веб-застосунок для онлайн-магазину книг, розроблений з використанням HTML, CSS та чистого JavaScript. Проєкт імітує повний цикл роботи інтернет-магазину, включаючи каталог товарів, кошик, систему відгуків та особистий кабінет користувача.

Всі дані (користувачі, кошик, відгуки) зберігаються локально у браузері (LocalStorage), що дозволяє тестувати функціонал без підключення до бекенду.

## Основний функціонал

- **Каталог книг**: Пошук, фільтрація (за ціною, жанром, рейтингом) та сортування.
- **Користувач**: Реєстрація, авторизація, редагування профілю.
- **Взаємодія**: Додавання книг у кошик та до списку обраних (Favorites).
- **Відгуки**: Можливість залишати оцінки та коментарі до книг.
- **Адаптивність**: Повна підтримка мобільних пристроїв та планшетів.

## Як запустити проєкт

Цей проєкт є статичним веб-сайтом, тому його можна запустити декількома способами:

### Спосіб 1: Прямий запуск (Найпростіший)
Знайдіть файл `index.html` у папці проєкту та відкрийте його подвійним кліком. Сайт відкриється у вашому веб-браузері за замовчуванням.

### Спосіб 2: Через розширення VS Code (Рекомендовано)
Якщо ви використовуєте редактор коду VS Code:
1. Встановіть розширення **Live Server**.
2. Відкрийте файл `index.html`.
3. Натисніть кнопку **Go Live** у нижньому правому куті вікна VS Code.
